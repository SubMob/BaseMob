## Install

BaseMob is currently published to Maven Central, so add that to repositories.

```groovy
repositories {
    mavenCentral()
}
```

Then, simply add the dependency to your common source-set dependencies

```groovy
commonMain {
    implementation("com.github.submob:basemob:LATEST_VERSION")
}
```
